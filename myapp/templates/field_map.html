Item Form

The most complex form in the app. 42 fields across create and edit flows, heavily driven by item type. 6 fields carry a 💡 context-aware suggestion button (issuer, logo slug, wallet, discount applied, currency, code type). Suggestions are ranked by recency/frequency, boosted when sibling fields share the same context (e.g. same issuer), with automatic cross-type fallback when fewer than 3 suggestions exist for the selected type. Accepted suggestions later changed before saving record a ScanFieldCorrection row to self-improve future rankings.
